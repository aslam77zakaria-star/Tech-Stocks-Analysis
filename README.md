## 📖 Overview

This project analyzes stock market data for leading technology companies and presents insights through an interactive dashboard.

The dashboard enables users to:

- 📊 Explore historical stock prices
- 📈 Analyze stock performance
- 📉 Compare multiple companies
- 📅 Filter data by date
- 📌 Generate interactive visualizations

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/aslam77zakaria-star/Tech-Stocks-Analysis.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
